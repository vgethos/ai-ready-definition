## Screen 4

**URL:** https://app.ethos.com/q/protections

### What the user sees
- **Header**: "ETHOS" logo centered at the top; back arrow (←) on the left; "NEED HELP? (415) 915-0665" on the right
- **Progress indicator**: A thin green progress bar near the top of the screen
- **Heading**: "Who depends on you financially?"
- **Subheading**: "This helps us customize your plan. Select all that apply."
- **Four selectable options**, each with an icon and label:
  1. Spouse or partner (icon: two figures)
  2. Children (icon: adult with child)
  3. Parent (icon: two figures, different style)
  4. Other (icon: heart/hands)

### What the user can do
- **← Back button**: Navigate to the previous screen
- **Phone number link** "(415) 915-0665": Call for help
- **"Spouse or partner" checkbox/radio**: Select this dependent type
- **"Children" checkbox/radio**: Select this dependent type
- **"Parent" checkbox/radio**: Select this dependent type
- **"Other" checkbox/radio**: Select this dependent type (multiple selections allowed per "Select all that apply.")
- **"Next" button** (bottom, full-width, gray/disabled state): Proceed to the next screen once at least one option is selected
