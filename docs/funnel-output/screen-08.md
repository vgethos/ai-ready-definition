## Screen 8

**URL:** https://app.ethos.com/q/children

### What the user sees
- **Header**: "ETHOS" logo centered at top; "NEED HELP? (415) 915-0665" in the top-right corner
- **Progress bar**: Partially filled green bar near the top, indicating progress through the funnel
- **Heading**: "How many children do you have under 18?"
- **Subtext**: "This helps us estimate your family coverage needs."
- **Answer options** (displayed as full-width selectable rows): 0 / 1 / 2 / 3 / 4+

### What the user can do
- **Back arrow** (top-left): Navigate to the previous screen
- **"NEED HELP? (415) 915-0665"** (top-right): Call Ethos support
- **"0" option button**: Select zero children under 18
- **"1" option button**: Select one child under 18
- **"2" option button**: Select two children under 18
- **"3" option button**: Select three children under 18
- **"4+" option button**: Select four or more children under 18
