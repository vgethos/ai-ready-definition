## Screen 6

**URL:** https://app.ethos.com/q/intent

### What the user sees
- **Header**: "ETHOS" logo centered at top; "NEED HELP? (415) 915-0665" in the top-right corner; a back arrow (←) in the top-left
- **Progress bar**: A partially filled green bar near the top, indicating early-to-mid funnel progress
- **Question heading** (large, bold): "When would you like to be covered by life insurance?"
- **Four answer options** displayed as full-width selectable tiles, each with light background:
  1. "I'm ready today"
  2. "Within a week"
  3. "In a few months"
  4. "I'm not sure"

### What the user can do
- **← Back button** (top-left): Navigate to the previous screen
- **NEED HELP? (415) 915-0665** (top-right): Tap to call support
- **"I'm ready today"** tile: Select this answer to indicate immediate coverage intent
- **"Within a week"** tile: Select this answer to indicate coverage needed within a week
- **"In a few months"** tile: Select this answer to indicate coverage needed in a few months
- **"I'm not sure"** tile: Select this answer if undecided on timing

Selecting any tile advances the funnel to the next step (no separate submit button).
